# fullstackopen_course
## 📚 Ejercicio 1.9 — Unicafe paso 4

**Enunciado clave:**  
Haz que las estadísticas solo se muestren si ya se ha dado algún feedback. Si no hay ningún comentario, en su lugar debe aparecer el texto:
`No feedback given`

**Lo esencial a aprender:**

- Renderizado condicional en React (`if`, operador ternario, retornos tempranos).  
- Evitar mostrar estadísticas vacías o erróneas.  
- Mejor experiencia de usuario al no mostrar datos innecesarios.
