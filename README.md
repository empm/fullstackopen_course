# fullstackopen_course
## 📚 Ejercicio 1.6 — Unicafe paso 1

**Enunciado clave:**  
Implementa una aplicación de feedback con tres botones (`good`, `neutral`, `bad`). Al hacer clic en cada botón se incrementa un contador respectivo. Muestra los valores de cada contador.

**Lo esencial a aprender:**

- Uso de `useState` para manejar estado local (tres estados distintos).  
- Cómo pasar funciones manejadoras (`onClick`) para actualizar estado.  
- Separar cada botón con su propia lógica de actualización.
