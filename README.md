# fullstackopen_course
## 📚 Ejercicio 1.7 — Unicafe paso 2

**Enunciado clave:**  
Agrega más estadísticas a la aplicación:  
- Total de comentarios  
- La puntuación promedio (donde `good = +1`, `neutral = 0`, `bad = –1`)  
- Porcentaje de comentarios positivos

**Lo esencial a aprender:**

- Cálculo de valores derivados del estado (suma total, promedio, porcentaje).  
- Manejar divisiones por cero cuando no hay comentarios.  
- Uso de `toFixed` para redondear decimales.  
- Concatenar el símbolo `%` para mostrar porcentaje.
